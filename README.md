# Chinmoy Build Develop

[![CI](https://github.com/Chinmoy-sh/Chinmoy-Build-Develop/actions/workflows/ci.yml/badge.svg)](https://github.com/Chinmoy-sh/Chinmoy-Build-Develop/actions/workflows/ci.yml)

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Overview

This repository contains a production-ready static portfolio site with:

- polished visual design and interaction effects
- responsive layout across desktop and mobile
- project showcase, testimonials, and contact flow
- repository standards for quality, collaboration, and CI

## Live Preview

You can run the site locally with any static server.

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Tailwind CSS (CDN)
- Three.js (CDN)

## Getting Started

### 1. Clone

```bash
git clone https://github.com/Chinmoy-sh/Chinmoy-Build-Develop.git
cd Chinmoy-Build-Develop
```

### 2. Install tooling

```bash
npm install
```

### 3. Run locally

```bash
npm run start
```

## Available Scripts

- `npm run start`: serve the site locally
- `npm run lint`: run HTML, CSS, and JavaScript lint checks
- `npm run format`: format supported files with Prettier
- `npm run format:check`: verify formatting without changing files
